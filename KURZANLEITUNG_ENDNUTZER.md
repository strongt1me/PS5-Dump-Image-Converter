# Kurz-Anleitung fuer Endnutzer

## Wofuer ist das Programm?

Das Programm hilft dir, PS5-Dateiformate umzuwandeln und Dumps zu pruefen.

## So startest du

1. Programm oeffnen
2. Aufgabe auswaehlen
3. Quelle waehlen
4. Zielordner waehlen
5. Start klicken
6. Warten bis Fertig-Meldung erscheint

## Die 8 Aufgaben (einfach erklaert)

1. Ordner zu ffpfsc
2. ffpfsc zu exFAT
3. exFAT zu ffpfsc
4. ffpfsc zu Ordner
5. exFAT zu Ordner
6. ffpkg zu ffpfsc
7. fakelib verwalten
8. Dump auf Fehler pruefen

## Hilfsmenu: Ressourcen (neu 19-21)

Wenn ein zusaetzliches Tool fehlt, oeffne Ressourcen und nutze:

- [19] OSFMount automatisch installieren
- [20] Dokan2 automatisch installieren
- [21] FileZilla automatisch installieren

Wichtig:

- Fehlende Tools werden automatisch heruntergeladen und installiert.
- Bereits installierte Tools werden nicht erneut installiert.
- Installation laeuft im Hintergrund.

## Haefige Probleme und schnelle Hilfe

1. Installation startet nicht
   - Programm als Administrator starten
   - Internetverbindung pruefen
2. Windows-Sicherheitsfenster (UAC)
   - Mit Ja bestaetigen
3. Antivirus blockiert
   - Ausnahme fuer das Programm erlauben
4. Falsche Datei ausgewaehlt
   - Endung pruefen (z. B. .ffpfsc, .exfat, .ffpkg)

## Kurz-Check vor Nutzung

1. Genug freier Speicher vorhanden
2. Quelle und Ziel korrekt gesetzt
3. Richtige Aufgabe ausgewaehlt
4. Bei Bedarf zuerst Ressourcen 19-21 nutzen

## Tipp

Wenn etwas nicht klappt: zuerst Aufgabe 8 (Validator) laufen lassen. Das zeigt oft schnell, wo das Problem liegt.
